## Building Executable:

just call: 'make'

if there are any permission issues with the executable, run 'make clean' and then 'make' again

## Running Executable:

run: 'simulation <number of CPU workers> <Value of S> <filename>'
ex. tasksV3 8 1600 tasks.txt

## Necessary files:

Simulation_Main.c
Queue_Support.c
Queue_Support.h
